# Components

คอมโพเนนต์ของโปรเจกต์ Next.js (Pages Router)

## กฎการตั้งชื่อไฟล์ (แนะนำ)

- **คอมโพเนนต์ React (.tsx):** ใช้ **PascalCase** เช่น `CaseStatusBadge.tsx`, `EvidenceManager.tsx`
- **ไฟล์ที่ไม่ใช่คอมโพเนนต์ (util, style):** ใช้ kebab-case ได้ตามเดิม เช่น `theme-switch.tsx`, `navbar.tsx`

เมื่อแก้หรือเพิ่มไฟล์ใหม่ แนะนำให้ใช้ PascalCase สำหรับคอมโพเนนต์เพื่อความสม่ำเสมอ
